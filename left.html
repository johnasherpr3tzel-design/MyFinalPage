Cybersecurity Fundamentals

Confidentiality - Confidentiality is about a disclosure or a security breach of the data. it refers to the practices and measures that ensure sensitive information is accessible only to authorised individuals. Often confidentiality is compromised by cracking poorly encrypted data, Man-in-the-middle (MITM) attacks, disclosing sensitive data.

Integrity - Integrity refers to protecting information from being modified by unauthorized parties. it's known as Data Integrity for most people it was ensured that you were protected by an unknown user which is a high risk Examples of Data Integrity is:

  • Email Fraud Attacks
  • Financial Fraud
  • Manipulation of Healthcare Records
  • Malware Targeting System files or firmware

Availability - it's a type of attack that making sure that attackers have authorized access to the user's information. it also means your systems and data must be accessible when needed, whether during normal operations or in the face of a cyber attack, power outage, or natural disaster. Example of Availability Threats is:

  • Denial of Service
  • Ransomware
  • Software and Website Bugs

Types of Cyberattacks

there are two types of cyberattacks

  • Web Based Attacks
  • System Based Attacks

Web Based Attacks - A web browser-based attack targets vulnerabilities in web browsers to compromise user data or execute malicious code. These attacks also exploit weaknesses in web applications accessed via web browsers, compromising user data, stealing sensitive information, or disrupting services.

Injection attacks - It is the attack in which some data will be injected into a web application to manipulate the application and fetch the required information. DNS Spoofing - DNS Spoofing is a type of computer security hacking. Session Hijacking - It is a security attack on a user session over a protected network. Phishing - Phishing is a type of attack which attempts to steal sensitive information. Brute force - It is a type of attack which uses a trial-and-error method. Denial of Service - It is an attack which meant to make a server or network resource unavailable. Dictionary attacks - This type of attack stored the list of a commonly used password. URL Interpretation - It is a type of attack where we can change the certain parts of a URL. File Inclusion attacks - It is a type of attack that allows an attacker to access unauthorized files. Man in the middle attacks - It is a type of attack that allows an attacker to intercepts the connection.

System Based Attacks - the type of attack that compromises your computer the System based attacks is different than Web Based Attacks the System Based Attacks only attacks the computer network or server Here are the example of the System Based Attacks:

  1. Virus
  2. Worm
  3. Trojan horse
  4. Backdoors
  5. Bots

Security Vunerabilities or Threat of Attacks

Categories of vulnerabilities

  • Corrupted (Loss of integrity)
  • Leaky (Loss of confidentiality)
  • Unavailable or very slow (Loss of availability)

Threats represent potential security harm to an asset when vulnerabilities are exploited. Attacks are threats that have been carried out

  • Passive – Make use of information without affecting system resources
  • Active – Alter system resources or affect operation
  • Insider – Initiated by an entity inside the organization
  • Outsider – Initiated from outside the perimeter
